### 网站定位

聚财众发网站是一个基于Astro框架构建的现代化金融资讯网站，专注于提供专业的投资分析、市场洞察和播客内容。网站旨在为高净值投资者和理财爱好者提供高质量的金融信息服务，帮助用户实现资产增值目标。

### 1.1 网站定位
- 目标用户：高净值投资者、理财爱好者
- 核心功能：提供专业投资分析文章、播客内容、投资策略分享
- 网站特色：深度行业研究、专业投资洞察、多元化内容形式

### 1.2 技术架构
- 前端框架：Astro (最新版本)
- UI框架：Tailwind CSS
- 组件库：React 及原生 Astro 组件
- 部署平台：Vercel (支持自动构建和部署)
- 内容管理：Markdown + Astro Content API
- 编程语言：TypeScript, JavaScript
- 构建工具：Vite
- 图片优化：Sharp, 自定义压缩脚本
- 音频处理：Web Audio API
- API支持：Serverless Functions

## 三、使用指南

### 3.1 浏览文章
1. 进入文章列表页：点击导航栏中的「研究文章」
2. 筛选文章：点击页面上方的专栏按钮（全部文章、日斗专栏、聚财策略）
3. 阅读文章：点击感兴趣的文章卡片，进入详情页阅读全文

### 3.2 收听播客
1. 进入播客列表页：点击导航栏中的「聚财播客」
2. 选择播客：点击感兴趣的播客卡片
3. 播放音频：在播客详情页，点击音频播放器的播放按钮
4. 阅读文字稿：音频下方提供播客文字稿，可同步阅读

### 3.3 使用智能客服小聚2.0
1. 打开客服窗口：点击网站右下角的蓝色聊天图标
2. 基础咨询：在输入框中输入您的问题，或点击快捷回复按钮
3. 文章管理：
   - 搜索文章：输入'搜索文章+关键词'（例如：搜索文章 投资策略）
   - 创建文章：输入'创建文章'，按照指引提供标题、描述和内容
   - 更新文章：输入'更新文章'，按照指引提供文章标题和新内容
   - 删除文章：输入'删除文章'，提供要删除的文章标题
4. 投资咨询：输入'投资咨询'，选择感兴趣的投资类型
5. 风险评估：输入'风险评估'，按照指引提供相关信息
6. 联系人工：如需人工服务，可在对话中输入'联系人工客服'

### 3.4 其他功能
- 播客详情页提供音频播放和文字稿阅读功能
- 您可以通过分享按钮将喜欢的播客分享给好友


## 四、内容管理

### 4.1 文章管理
- 文章存储位置：`src/content/articles/`
- 文章格式：Markdown
- 文件命名规则：采用`YYYY-MM-DD-文章标题.md`格式（例如：`2025-08-12-new-article.md`）
- 文章元数据：
  ```yaml
  ---
  title: 文章标题（必填）
  description: 文章描述（必填）
  publishDate: '发布日期'（必填，格式：YYYY-MM-DD，必须用单引号包裹）
  coverImage: 封面图路径（可选，默认使用`public/images-optimized/article-default.jpg`）
  column: 所属专栏（必填，可选值：日斗专栏、聚财策略）
  ---
  ```
- 完整文章示例：
  ```markdown
  ---
  title: 2025年市场投资新趋势分析
  description: 本文深入分析了2025年全球市场投资的主要趋势和机会，为投资者提供参考。
  publishDate: '2025-08-12'
  coverImage: /images-optimized/article-cover-1.jpg
  column: 日斗专栏
  ---

  # 2025年市场投资新趋势分析

  ## 一、全球经济概况

  2025年全球经济呈现复苏态势，但区域分化明显...

  ## 二、主要投资机会

  ### 2.1 科技板块

  人工智能、量子计算等新兴技术将继续引领增长...

  ### 2.2 新能源领域

  清洁能源转型带来长期投资机会...

  ## 三、风险提示

  投资者需关注地缘政治风险、通胀压力等因素...
  ```
- 注意事项：
  1. publishDate必须用单引号包裹，以确保系统正确解析为字符串
  2. 文章内容使用Markdown格式，支持标题、列表、表格等基本语法
  3. 图片请上传至`public/images-optimized/`目录，并在文章中使用相对路径引用

### 4.2 播客管理
- 播客存储位置：`src/content/podcasts/`
- 播客格式：Markdown
- 播客元数据：
  ```yaml
  title: 播客标题
  description: 播客描述
  publishDate: 发布日期
  coverImage: 封面图路径
  audioUrl: 音频文件路径
  column: 所属专栏
  ```

### 4.3 图片管理
- 原始图片存储：`public/images/`
- 优化后图片存储：`public/images-optimized/`
- 建议：上传图片前进行压缩优化，以提高网站加载速度

## 五、网站维护

### 5.1 开发环境搭建
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 访问本地网站：`http://localhost:4321`

### 5.2 生产构建
1. 构建生产代码：`npm run build`
2. 预览构建结果：`npm run preview`

### 5.3 部署
- 网站通过Vercel自动部署
- 提交代码到主分支后，Vercel会自动触发部署流程

### 5.4 常用维护命令
- 压缩图片：`node compress-images.cjs`
- 移除重复图片：`node remove-duplicate-images.cjs`
- 比较目录大小：`node compare-sizes.cjs`
